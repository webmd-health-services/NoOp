# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'NoOp'
Import-Module -Name 'NoOp'
```

To install privately:

```powershell
Save-Module -Name 'NoOp' -Path '.'
Import-Module -Name '.\NoOp'
```
